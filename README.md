## Руководство пользователя системы хранения файлов

### Краткое описание
Данная система предоставляет функционал для работы с файлами через командную строку.

---
### Проектирование БД

![Альтернативный текст](images/BD-Diagram.svg)


### Основные команды

* **register** `<username>` `<password>`
  Регистрация нового пользователя в системе

* **login** `<username>` `<password>`
  Авторизация в системе

* **logout**
  Выход из системы

* **upload** `<filePath>`
  Загрузка файла в систему

* **download** `<link>`
  Скачивание файла по ссылке

* **generate** `<fileId>` `<expiration>`
  Создание ссылки на файл с указанием времени действия

* **list**
  Просмотр списка всех файлов

* **delete** `<fileId>`
  Удаление файла из системы

* **help**
  Отображение справки по командам

* **exit**
  Завершение работы с системой

---

### Примеры использования

```
# Регистрация нового пользователя
register vladimir123 qwerty123

# Вход в систему
login vladimir123 qwerty123

# Загрузка файла
upload C:\\Users\\Documents\\file.txt

# Создание ссылки на файл
generate 123456789 60

# Скачивание файла
download abcdefghijklmno

# Выход из системы
logout

# Регистрация нового пользователя
register vladimir123 qwerty123

# Вход в систему
login downloadUser 123

# Скачивание файла с другого аккаунта
download abcdefghijklmno

```

---

### Важные замечания

* Перед загрузкой файлов необходимо авторизоваться
* Все команды чувствительны к регистру
* При работе с файлами используйте абсолютные пути
* Время действия ссылки указывается в минутах

---

### Ошибки и их обработка

* При неверном логине/пароле система сообщит об ошибке
* При отсутствии прав доступа к файлу появится соответствующее сообщение
* При неверном формате команды система покажет подсказку

---

### Дополнительная информация

Для получения справки в любой момент введите команду **help**

Система автоматически сохраняет все изменения и обеспечивает целостность данных.
